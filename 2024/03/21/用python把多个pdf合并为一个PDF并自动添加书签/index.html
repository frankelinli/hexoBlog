这是以前的一个python小程序。假如文件夹下有5个PDF文件，他们的名字分别是：

  1. hello
  2. world
  3. nihao
  4. shijie
  5. thankyou

把他们合并到成一个jieshao.pdf文件，并且用上面5个文件的名字来作为jieshao.pdf的书签。

:::info

这个函数会遍历要求目录下的所有文件，将以 .pdf 结尾的所有pdf合并成一个新的pdf，并保存在 file_dir 目录下

:::
